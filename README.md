# Library Management System (SQL Project)
# Description
This project is a Library Management System built using SQL. It manages students, books, and book issue records.
## Tables Used
* Students
* Books
* Issue
## Features
* Add students and books
* Issue books to students
* Track returned and not returned books
* Count books issued per student
* Find most active student
## Tools Used
* MySQL
* GitHub
